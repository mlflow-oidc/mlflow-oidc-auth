Create `mlflow_oidc_auth/repository/_base.py` with 4 generic base classes. Each uses `typing.Generic` with `TypeVar` for the SQLAlchemy model and entity classes.

**Design approach:** The base classes provide the *common skeleton* for CRUD operations. Subclasses set class attributes for their specific model, entity, and column names. Methods that vary between repo types (e.g., scorer's 2-part key) are simply overridden in subclasses.

**File content for `mlflow_oidc_auth/repository/_base.py`:**

```python
"""Generic base classes for permission repositories.

Provides common CRUD patterns for:
- BaseUserPermissionRepository: user-level permissions (resource_id + username)
- BaseGroupPermissionRepository: group-level permissions (resource_id + group_name)  
- BaseRegexPermissionRepository: regex-based user permissions (regex + priority + username)
- BaseGroupRegexPermissionRepository: regex-based group permissions (regex + priority + group_id)
"""

from typing import Callable, Generic, List, Type, TypeVar

from mlflow.exceptions import MlflowException
from mlflow.protos.databricks_pb2 import INVALID_STATE, RESOURCE_ALREADY_EXISTS, RESOURCE_DOES_NOT_EXIST
from sqlalchemy.exc import IntegrityError, MultipleResultsFound, NoResultFound
from sqlalchemy.orm import Session

from mlflow_oidc_auth.db.models import SqlUser, SqlGroup
from mlflow_oidc_auth.permissions import _validate_permission
from mlflow_oidc_auth.repository.utils import get_user, get_group, list_user_groups, validate_regex

ModelT = TypeVar("ModelT")
EntityT = TypeVar("EntityT")
```

**Class 1: `BaseUserPermissionRepository(Generic[ModelT, EntityT])`**

Class attributes that subclasses must set:
- `model_class: Type[ModelT]` â€” e.g., `SqlExperimentPermission`
- `resource_id_attr: str` â€” column name on the model, e.g., `"experiment_id"`, `"name"`, `"endpoint_id"`

Methods:
- `__init__(self, session_maker: Callable[[], Session])` â€” stores `self._Session = session_maker`
- `_get_permission(self, session: Session, resource_id: str, username: str) -> ModelT` â€” query with join on SqlUser, filter by `getattr(self.model_class, self.resource_id_attr) == resource_id` and `SqlUser.username == username`, `.one()`. Raises `MlflowException(RESOURCE_DOES_NOT_EXIST)` on NoResultFound, `MlflowException(INVALID_STATE)` on MultipleResultsFound.
- `grant_permission(self, resource_id: str, username: str, permission: str) -> EntityT` â€” _validate_permission, get_user, create model instance with `{self.resource_id_attr: resource_id, "user_id": user.id, "permission": permission}`, add+flush, return `to_mlflow_entity()`. Catch IntegrityError â†’ RESOURCE_ALREADY_EXISTS.
- `get_permission(self, resource_id: str, username: str) -> EntityT` â€” session â†’ `_get_permission` â†’ `to_mlflow_entity()`
- `list_permissions_for_user(self, username: str) -> List[EntityT]` â€” get_user, query.filter(model.user_id == user.id)
- `list_permissions_for_resource(self, resource_id: str) -> List[EntityT]` â€” query.filter(resource_id_attr == resource_id)
- `update_permission(self, resource_id: str, username: str, permission: str) -> EntityT` â€” _validate_permission, _get_permission, set permission, flush
- `revoke_permission(self, resource_id: str, username: str) -> None` â€” _get_permission, delete, flush

**Class 2: `BaseGroupPermissionRepository(Generic[ModelT, EntityT])`**

class ExperimentPermissionRepository(BaseUserPermissionRepository[SqlExperimentPermission, ExperimentPermission]):

**Specific resource_id_attr values per repo:**

- RegisteredModelPermission: `resource_id_attr = "name"`
- GatewayEndpointPermission: `resource_id_attr = "endpoint_id"`

Add `create = grant_permission` and `delete = revoke_permission` after class body, OR override to preserve existing method names. **Choose: keep existing method names as aliases to base methods** for backward compatibility.
